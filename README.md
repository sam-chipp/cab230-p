## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/sam-chipp/cab230-p.git
   ```
2. Create SQL database using the provided stocks SQL.

3. Install NPM packages from frontend/backend directories
   ```sh
   npm install
   ```
4. Requires /etc/ssl/private/node-selfsigned.key to run backend 

5. Run npm start from frontend/backend
  ```sh
    npm start
  ```
